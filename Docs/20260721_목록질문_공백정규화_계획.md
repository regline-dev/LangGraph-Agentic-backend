# 2026-07-21 — 목록 질문 공백 정규화

## 문제
- `전체 목록`(띄어쓰기), `목록`(단독)이 catalog에 안 걸려 vague 되묻기로 빠짐
- 합의: 목록 의도면 제목 전량 나열

## 해결
- catalog **마커 비교용** 문자열만 공백 제거 (`전체 목록` → `전체목록`)
- 마커에 단독 `목록` 추가
- 제목/다른 라우터에는 공백 제거 적용하지 않음

## 완료 기준 체크리스트
- [x] `전체 목록` → 제목 번호 목록
- [x] `목록` → 제목 번호 목록
- [x] `우화목록은` 기존 통과 유지
- [x] `제일 재밌는…` fun rank 유지 (회귀 없음)
- [x] pytest 관련 스위트 통과 (`test_human_test_fixes` 13)
- [x] `키워드 목록`은 catalog 미매칭 (오탐 방지)
